# Research Archive

This directory stores full point-in-time stock analysis reports.

Recommended path pattern:
- `YYYY/MM/DD/<ticker>-analysis.md`

Each report should capture:
- company and ticker
- analysis date
- point-in-time price
- current valuation snapshot
- recent news with interpretation
- bear/base/bull assumptions
- fair value outputs
- provisional conclusion
- monitoring items

NotebookLM should receive only the distilled reasoning and reusable lessons, not every full report verbatim.
